# ifes-java-exercicios

Exercícios de lógica de programação em Java desenvolvidos durante meus estudos no IFES.

## Conteúdo
- Sequência de Fibonacci (verificação se um número pertence à sequência)
- MMC (Mínimo Múltiplo Comum) utilizando fatoração
- Fórmula de Bhaskara com cálculo da raiz pelo método babilônico

## Organização
- `basico/` — exercícios introdutórios
- `matematica/` — exercícios matemáticos

## Tecnologias
- Java
- Scanner para entrada de dados
- Estruturas de repetição e condicionais

## Observação
Projeto com foco em aprendizado, prática de lógica e organização de código.
